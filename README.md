# 🛒 Store - Fake Store Web Application

A modern and responsive e-commerce frontend built with vanilla JavaScript, HTML, and CSS. This project demonstrates a complete user flow including authentication, product browsing, filtering, and admin dashboard.

## ✨ Features

- **Authentication System**
  - User login with validation
  - Cookie-based authentication
  - Protected routes (Dashboard)

- **Product Management**
  - Browse all products from Fake Store API
  - Real-time search functionality
  - Category filtering (All, Electronics, Jewelery, Men's Clothing, Women's Clothing)
  - Product cards with rating and price

- **User Dashboard**
  - View list of all users
  - Detailed user information (Name, Username, Email, Phone, Address)

- **Responsive & Modern UI**
  - Clean and user-friendly design
  - Hover effects and smooth transitions
  - Mobile-friendly layout

## 🛠️ Technologies Used

- **HTML5** & **CSS3**
- **Vanilla JavaScript (ES6+)**
- **Fetch API** for backend communication
- **Fake Store API** (`https://fakestoreapi.com`)
- **Font Awesome** icons
- **CSS Variables** for theming
